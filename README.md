# Onyx Orbit Dining 🌌

**A State-of-the-Art Digital Sensory Experience**

Onyx Orbit Dining is an immersive, high-end restaurant website built with a "Luxury Noir" aesthetic. It merges cutting-edge web technologies to create a cinematic journey through the artistry of fine dining.

---

## ✨ Features

- **3D "Living Dish" Hero:** A hyper-realistic, procedurally detailed 3D focal point featuring custom **GLSL Glisten Shaders** that react to light and mouse movement.
- **Kinetic Scroll Narrative:** A cinematic camera journey powered by **GSAP ScrollTrigger**, gliding through the dish and its ingredients as the user explores the story.
- **Tactile UI (Barely-There):** A minimalist interface featuring a **Magnetic Cursor**, animated navigation, and a grid-based **Menu Overlay**.
- **Sensory Polish:** High-end post-processing (Depth of Field/Bokeh, Chromatic Aberration, and Analog Film Grain) for a professional photography feel.
- **Silky Motion:** Integrated **Lenis** smooth-scrolling for a consistent, premium feel across all browsers and devices.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router, TypeScript)
- **3D Engine:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) & [Drei](https://github.com/pmndrs/drei)
- **Animation:** [GSAP (GreenSock)](https://gsap.com/)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Shaders:** Custom GLSL (Fragment & Vertex)
- **Styling:** CSS Modules (Vanilla)

## 🚀 Getting Started

### 1. Local Development
```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the result.

### 2. GitHub Pages Deployment
This project is configured for automatic deployment via **GitHub Actions**.

1. Ensure your repository name matches the `basePath` in `next.config.ts`.
2. Push your changes to the `main` branch.
3. Enable GitHub Pages in your repo settings (**Settings > Pages > Build and deployment > Source: GitHub Actions**).
