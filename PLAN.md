# Plan: High-End Interactive Restaurant Website

## Objective
Create a state-of-the-art, immersive restaurant website using Next.js, Three.js (R3F), and GSAP. The goal is a "digital sensory experience" that rivals award-winning sites like *The Alchemist* or *Mirazur*.

## Inspiration & Aesthetic
*   **Aesthetic:** "Luxury Noir" or "Minimalist Zenith." Dark background, premium typography (serif headers, sans-serif body), high-contrast visuals, and generous white space (or "black space").
*   **Key Inspiration:** 
    *   *The Alchemist:* For immersive atmosphere and storytelling.
    *   *Mirazur:* For organic, scroll-triggered narrative flow.
    *   *Zenith:* For "Barely-There UI" that focuses on 3D products.

## State-of-the-Art Features
1.  **3D Hero "Living Dish":** A hyper-realistic 3D model of a signature dish that features "breathing" shaders (subtle steam, glistening textures).
2.  **Kinetic Scroll Narrative:** As the user scrolls, the 3D scene doesn't just rotate—it transforms. Camera paths glide through the ingredients, and lighting shifts from "Day" to "Night" (Dinner service).
3.  **The "Fluid Menu":** A WebGL-based menu transition where clicking an item "liquifies" the screen or uses a 3D "Flip" effect to reveal details.
4.  **Custom GLSL Shaders:** 
    *   *Bokeh/Depth of Field:* Dynamically shifting focus based on scroll.
    *   *Grain & Bloom:* For a cinematic, analog feel.
5.  **Micro-Sound UX:** Ultra-subtle audio cues (clink of glass, soft kitchen ambiance) triggered by user interaction.

## Technical Stack
*   **Framework:** Next.js 14+ (App Router, React 18+).
*   **3D Engine:** React Three Fiber + Drei + Postprocessing.
*   **Animation:** GSAP (ScrollTrigger, Flip, Observer) + Lenis for smooth scrolling.
*   **Styling:** Vanilla CSS Modules + Tailwind for layout.
